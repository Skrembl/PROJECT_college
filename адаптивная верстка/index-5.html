HTML

HTML (HyperText Markup Language) — это язык разметки, который используется для создания веб-
страниц. С помощью HTML определяется структура содержимого документа, такого как заголовки,
абзацы, списки, ссылки и медиаэлементы. HTML использует теги для определения различных элементов
страницы. Каждый тег представляет собой открывающийся и закрывающийся элемент, который
окружает определенное содержимое.
HTML обеспечивает основу для создания веб-страниц, определяя их структуру и содержимое. Он
работает с другими технологиями, такими как CSS (Cascading Style Sheets) для стилизации страницы и
JavaScript для создания интерактивных элементов. HTML является основой веб-разработки и является
необходимым языком для любого веб-разработчика.

К примеру, <р><р> - выделяет абзац на web-странице
